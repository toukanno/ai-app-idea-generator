# AI献立マネージャー「MealBrain」

## 概要
冷蔵庫の中身をAIが認識し、栄養バランスを考慮した献立を自動提案するアプリです。

## 主な機能
- **食材スキャン**: カメラで冷蔵庫を撮影するだけで食材を自動認識
- **栄養バランス分析**: 家族構成に合わせた栄養素の過不足を可視化
- **レシピ自動生成**: 在庫食材だけで作れるレシピをAIが提案
- **買い物リスト連携**: 不足食材を自動でリスト化、近隣スーパーの特売情報と連動
- **食品ロス削減**: 賞味期限が近い食材を優先的に使うレシピを提案

## 技術スタック
- フロントエンド: React Native
- バックエンド: Node.js + PostgreSQL
- AI: 画像認識モデル + LLM
